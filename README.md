# SaleApp
BÀI THỰC HÀNH MÔN CÔNG NGHỆ PHẦN MỀM - THẦY NGUYỄN TRUNG HẬU
SaleApp DH23CS02 (HK1 - 2025 - 2026)

Cách chạy web
1. Clone link github về bằng câu lệnh : git clone https://github.com/Teaching2504/SaleApp.git
2. Mở MySQL Workbench để tạo cơ sở dữ liệu saledb (Lưu ý phải mở tab cơ sở dữ liệu để có thể kết nối với ứng dụng)
3. Mở code (dùng Pycharm) cài đặt môi trường venv ảo
4. Run file models.py và check các bảng cơ sở dữ liệu đã được cập nhật
5. Run file index.py
